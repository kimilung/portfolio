_NAME

Aspetti di Italia

_TYPE

Print Editorial Design

_OBJECTIVE

The objective of this project was to create a ficticious magazine that included: a cover with masthead, table of contents, and interior article. Learning to pair appropriate typefaces and imagery based on content, this project helped to build knowledge around proper design decisions. Design decisions included: typography, grid systems, hierarchy, and file organization.

_CLIENT

School Project

_TECHNOLOGY

Adobe InDesign, Lettering, Photoshop
